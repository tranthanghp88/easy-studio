# Easy Vietnamese Voice Studio

App gen voice tiếng Việt tách biệt hoàn toàn.

## Chạy app
1. Giải nén zip
2. Chạy `START_APP.bat`
3. Nhập API key Gemini hoặc cấu hình Vertex AI trong phần Nâng cao

## Hotkey
- `Ctrl + Shift + K`: mở Console DevTools

## Có sẵn
- 3 mode: Quảng cáo, Đọc truyện, Review phim
- Preview 20 giây
- Gen full audio
- Cache theo chunk
- BGM + auto ducking bằng ffmpeg
- Xuất WAV
