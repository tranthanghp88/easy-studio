# Easy Studio v33

- Removed wavebar peak marker dots.
- Easy Voice Việt now forces Gemini-only generation and no longer requires Vertex token/project/location.
- Native dialog titles are normalized to Easy Studio across the Shell.
- Added karaoke-style subtitles: speaker base colors remain, currently spoken words highlight white with subtle glow/scale.

Notes:
- Karaoke timing is approximate when word-level timestamps are unavailable: each subtitle cue duration is split across its words.
- Voice Việt still keeps the Key Manager UI, but Vertex generation is disabled/ignored.
